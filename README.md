# The Talismans of SMMA - Dark-Themed Website

A modern, dark-themed responsive website for a Social Media Marketing Agency (SMMA) built with HTML, CSS, and JavaScript featuring dynamic theme switching and enhanced scroll animations.

## 🌟 New Features

- **Dark Theme by Default**: Sleek dark design for modern appeal
- **Theme Toggle**: Users can switch between dark and light themes
- **Enhanced Animations**: Bidirectional scroll animations that work when scrolling up or down
- **Improved Branding**: "The Talismans of SMMA" with mystical, professional appeal

## 🔧 Customization

### Changing Themes
The website automatically starts in dark mode. Users can toggle between themes using the moon/sun icon in the navigation. Theme preference is automatically saved.

### Updating Content
- Edit text content directly in `index.html`
- Replace "The Talismans of SMMA" with your brand name
- Update contact information in the contact section

### Customizing Colors
Update the CSS variables in `styles/main.css`:
```css
:root {
  --gradient-primary: linear-gradient(135deg, #your-color 0%, #your-color-2 100%);
  --accent-color: #your-accent-color;
}

[data-theme="dark"] {
  --bg-primary: #your-dark-bg;
  --text-primary: #your-dark-text;
}
```

### Adding Images
- Replace icon placeholders with actual images
- Add portfolio images in the portfolio section
- Update team photos in testimonials
